# 散戶投資儀表板 — 專題計劃書

> 建立時間：2026-04-27
> 狀態：🟡 規劃中（Week 0）
> 截止日期：約 2026-06-08（6 週）

---

## 專題概述

一個有質感、有別於市面投資網站的散戶投資儀表板。
目標：課堂展示 / 專題評分。

---

## 技術棧

| 層 | 技術 |
|---|---|
| 前端 | Next.js + Tailwind CSS + shadcn/ui |
| 後端 | Python FastAPI |
| 資料庫 | SQLite（開發）→ PostgreSQL（選擇性） |
| 部署 | Vercel（前端）+ Railway（後端） |
| OCR | Claude API（對帳單圖片解析） |
| 股價資料 | yfinance（美股）、TWSE API（台股）、CoinGecko API（加密貨幣） |

---

## 三大功能模組

### ① 財報爬取 + 呈現
- [ ] 股票搜尋介面（輸入代碼）
- [ ] 台股財報抓取（公開資訊觀測站）
- [ ] 美股財報抓取（yfinance / SEC EDGAR）
- [ ] 損益表視覺化（折線圖、長條圖）
- [ ] 資產負債表視覺化
- [ ] 現金流量表視覺化
- [ ] 多期比較功能

### ② 資產負債表（多市場）
- [ ] 手動持倉輸入介面（股票代碼 + 數量 + 成本）
- [ ] 圖片上傳 → Claude API OCR 自動解析對帳單
- [ ] 台股即時報價串接（TWSE）
- [ ] 美股即時報價串接（yfinance）
- [ ] 加密貨幣報價串接（CoinGecko）
- [ ] 資產總覽儀表板（總市值、損益、報酬率）
- [ ] 資產配置圓餅圖（市場/類型分布）
- [ ] 個別持倉損益明細表

### ③ 市場情緒 + 工具
- [ ] 恐懼貪婪指數（加密）
- [ ] 大盤情緒指標（VIX）
- [ ] RSI 指標計算
- [ ] MACD 指標計算
- [ ] 財經新聞摘要（RSS 爬取）
- [ ] 股票快速搜尋工具

## 關鍵決策記錄

| 日期 | 決策 | 原因 |

## 資料來源清單

| 資料 | 來源 | 類型 |
|------|------|------|
| 台股即時報價 | TWSE OpenAPI | 免費 |
| 台股財報 | 公開資訊觀測站 | 爬蟲 |
| 美股報價 + 財報 | yfinance | 免費套件 |
| 加密貨幣報價 | CoinGecko API | 免費 |
| 市場情緒（恐貪指數） | alternative.me API | 免費 |
| 財經新聞 | RSS（Yahoo財經、鉅亨網） | 爬蟲 |
| 對帳單 OCR | Claude API | 付費（依用量） |
