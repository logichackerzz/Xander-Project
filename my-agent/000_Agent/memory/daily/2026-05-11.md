## 關鍵技術筆記

- **TWT44U 欄位**：col[3]=融資金額(NT$千), col[4]=融券金額(NT$千)，同單位可直接算比值
- **BFI82U 外資判斷**：條件要 `"外陸資" in name and "自營商" not in name`
- **Python 型別提示**：`tuple[str, str]` 需 Python 3.9+，目前環境是 3.8，要拿掉 return type annotation
- **LayoutShell z-index 最終方案**：blobs=fixed z-0 / Topbar wrapper=z-50 / main=z-10
