## Session 002 — PowerShell 啟動目錄修正

### 問題
每次開啟 Claude Code 都停在 `C:\Users\mybab` 而非 `C:\Xander-agent`。

### 解決方式
在 PowerShell 設定檔加入 `claude` wrapper function：
```powershell
$env:CLAUDE_CODE_NO_FLICKER = "1"

function claude {
    Set-Location 'C:\Xander-agent'
    & "C:\Users\mybab\.local\bin\claude.exe" @args
}

function cc {
    Set-Location 'C:\Xander-agent'
    & "C:\Users\mybab\.local\bin\claude.exe" @args
}
```

### 關鍵路徑
- Claude 執行檔：`C:\Users\mybab\.local\bin\claude.exe`
- PowerShell 設定檔：`C:\Users\mybab\OneDrive\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1`
- `.claude` 設定資料夾：`C:\Users\mybab\.claude`

### 結論
輸入 `cc` 即可啟動 Claude Code 並自動切換至 `C:\Xander-agent`。`CLAUDE_CODE_NO_FLICKER=1` 消除啟動閃爍。
