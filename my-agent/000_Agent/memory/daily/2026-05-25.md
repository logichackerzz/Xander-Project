## 一、今天解決的 Bug

### 1. node.exe 不斷累積問題
- 原因：舊 SOP 只 kill port 8000（後端），沒 kill 3000（前端），每次重啟都疊加 node 進程
- 修正：SOP 改成啟動前先同時 kill 3000 和 8000，確保乾淨重啟

### 2. 投影片 port 混淆
- 記憶裡誤記 slides dev server 是 port 4321
- 實際確認 `vite.config.js` 寫的是 **port 5173**
- 已修正記憶

### 3. 1.5 頁視覺錯誤（節間縫隙）
- 原因：相鄰 section 間的 marquee 跑馬燈帶（高度 ~48px）造成快速 snap 時看到一條縫
- 修正：把 index.html 裡 section 之間的 4 個 marquee HTML 元素全部刪除（CSS 保留備用）

### 4. FOLIO 5 個字母卡在畫面正中
- 原因：`.pt-l` CSS 沒有 `opacity: 0`，5 個字母預設顯示在中央
- 修正：加上 `opacity: 0`，`__folioSweep` 動畫開始前先設 `opacity: 1`，結束後 `.set(letters, { opacity: 0 })`

### 5. Hero "Folio" 無法飛出容器
- 原因：`.s__inner` 和 `.hero-title-wrap` 都有 `overflow: hidden`，title 飛不出去
- 修正：`exit()` 裡暫時設 `style.overflow = 'visible'`，`reset()` 裡恢復 `'hidden'`

## 二、過場動畫系統架構（今日最大產出）

### 核心設計哲學
> 「過場本身就是內容，不只是換頁效果。」

S01 → S02 是同一個詞「FOLIO」的三種形態：
1. **Hero 手寫字**（Dancing Script，S01 靜態）
2. **Hero 手寫字飛走**（過場本身就是動畫）
3. **F O L I O 散落大字**（S02 進場，同個字，不同狀態）

### Strategy A — 自訂 exit（S01、S02）
- Section 自帶 `exit(dir)` 函數
- exit 動畫本身就是視覺過場，不需要 overlay
- main.js 在動畫執行中途（mid-flight）fire instant scroll
  - S01 exit：scroll at t=0.28（title 已飛一半）
  - S02 exit：scroll at t=0.32（字母已散開）

### Strategy B — 輕量 bg flash（S03–S08）
- 這些 section 沒有自訂 exit
- `.pt-bg` 白色蓋板短暫 opacity 1→0，遮住 instant scroll 的跳動

### FOLIO overlay（保留備用）
`window.__folioSweep(dir)` — 5 個大字母從螢幕一側橫掃而過。
**目前不使用於常規過場**，保留給未來「slogan 頁」使用。

## 三、S01 Hero 動畫系統

### Enter 序列
1. `hero-eyebrow`：clip-path 左→右 wipe（0.7s）
2. `hero-title` "Folio"：從 yPercent:110 滑入（1.1s，expo.out）
3. `hero-line`：scaleX 0→1 從左往右劃（0.9s）
4. `hero-tagline`：blur(16px)→0 + scale 0.92→1（0.85s），同時觸發 scramble 特效
5. sticker badge：scale 0.6→1 + back.out 彈跳（0.9s），完成後開始旋轉

### Exit（Title IS the transition）
1. sticker / tagline / eyebrow / line：快速消失（0.14–0.18s）
2. "Folio" 標題：
   - 向前：x → `+130vw`，y → -18px（power3.in，0.44s）
   - 向後：x → `-130vw`，y → +18px

**主題**："What is Folio?" — 問句做標題，去除所有中文

**佈局**：5 個超大字母散落在整個視口，badge 穿插其中


字母尺寸：
- F: clamp(110px, 17vw, 230px)，旋轉 -8deg
- O: clamp(100px, 15vw, 200px)，旋轉 +7deg
- L: clamp(130px, 19vw, 260px)，旋轉 -14deg（最大、錨點）
- I: clamp(90px, 13vw, 175px)，旋轉 +5deg
- O: clamp(115px, 17vw, 230px)，旋轉 -9deg

**Badge 設計**：
- A：旋轉圓環，文字 `MARKETS · READABLE · FOLIO ·`，中央 ◎ 符號
- B：indigo 膠囊 pill，文字 `AI × Human`
- C：violet 年份印章，文字 `2026`

**Enter 動畫**：
1. "What is Folio?" clip-path wipe（0.65s）
2. 字母從上方掉落 + blur 清除 + back.out(1.7) 彈跳，stagger 0.09s
3. Badges back.out(2.2) 彈入
4. 之後每個字母開始獨立的 yoyo float（不同 dy/dr/dur/delay，感覺像漂浮）

**Exit 前進（字母大散場）**：
- F → 左上 (-65vw, -50vh)
- O → 右上 (+75vw, -35vh)
- L → 下方 (-28vw, +72vh)
- I → 左下 (-80vw, +55vh)
- O → 右下 (+85vw, +65vh)

**Exit 後退**：字母反重力飛回上方

## 五、簡報完整需求清單（給下次 session 用）

### 視覺風格要求
- **基調**：editorial 雜誌感 × GSAP 官網互動感，不是企業 PPT
- **配色**：完全沿用 Folio 配色（Indigo / Cyan / Violet / Emerald，亮色底 `#FAF9F6` 系列）
- **字體**：
  - Folio 手寫標題：Dancing Script 700
  - 大字母散落：Plus Jakarta Sans 800（`var(--f-naming)`）
  - Body / eyebrow：Space Grotesk（`var(--f-display)`）
  - 數字 / code / 標籤：IBM Plex Mono（`var(--f-mono)`）
- **漸層**：Indigo → Violet → Indigo-lt，135deg（所有大字、pt-l 共用）
- **背景策略**：大部分頁面用 `#FAF9F6`，Aurora 光球做深度

### 動畫風格要求
- **每頁都要有進場動畫**，不是靜態換頁
- **過場要有連貫性**，前後頁在視覺上有關聯（S01 Folio 飛走 → S02 F O L I O 落下）
- **GSAP 官網風格**：clip-path reveal、back.out 彈跳、scale+blur 霧化進場
- **貼紙/badge 元素**：GSAP 官網風格，穿插在版面中，有旋轉動畫
- 字母 enter：永遠用 `back.out` + `blur → 0`
- 字母 exit forward：散場（scatter 到各角落）
- 字母 exit backward：反重力飛回
