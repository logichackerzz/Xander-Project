## 反思 ── 期末報告案例：「API 卡牆，爬蟲解圍」

### 問題
需要兩個指標：市場廣度（S&P 500 成分股高於 MA50 比例）、CBOE 全美股期權多空比

### 嘗試一：Yahoo Finance ticker（失敗）
`^SPXA50R`、`^PCCE` 等全部 404 或 empty，yfinance 不支援廣度/衍生指標 ticker。

### 嘗試二：CBOE CDN CSV（失敗）
加完整 User-Agent + Referer 仍回 403，IP 地區封鎖。

### 解法：爬蟲 + 自行計算
- **市場廣度** → 爬 Wikipedia S&P 500 名單（需 User-Agent）+ yf.download() 批次 503 支，算 MA50 比例，10 分鐘 cache
- **Put/Call Ratio** → yfinance OI 非盤中為 0，改用 volume；合併 SPY + QQQ + IWM 所有到期日

### 關鍵技術學習
- API 被擋先試爬蟲，不要繞圈測同類 API
- yfinance options OI ≠ volume，非盤中用 OI 會拿到 0
- `pd.read_html()` 傳 `StringIO(html)` 而非原始字串
- Windows cp950 print Unicode exception 會 crash，先 `str(e)[:120]` 截斷
- TAIFEX 三大法人表：首列 15 欄（含序號+契約），後續列 13 欄，所有月淨額欄位不同位置
