# Xander 的 AI 能幹清單

> 這份清單會跟著「外部工具整合計畫」長大。每次整合完一個工具，都回來這裡加上它能做什麼。
> 目的：讓你（和未來的 AI session）一眼看到「我的 AI 現在到底會哪些具體動作」。

## 🟢 已整合的工具

### 🌐 Firecrawl（MCP 路線｜整合日期：2026-04-27）

- 抓取網頁內容、整理文章重點
- 比較多個網頁的資訊（例如：產品規格、政策條文）
- 快速把網頁資料餵給 AI 做分析，不用自己寫爬蟲
- ⚠️ 免費方案有每月額度限制，額度用完需升級或等下個月重置

**試試看的指令**（重啟 Claude Code 後）：
- 「幫我抓這個網頁的重點：[網址]」
- 「幫我整理這篇文章的核心論點：[網址]」
